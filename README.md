
  # Solopreneur OS 2026

  This is the codebase for **Solopreneur OS 2026** — a Vite + React + Firebase app for solopreneurs and freelancers. It includes:

  - A multi-currency finance tracker (income, expenses, overdue, bad debt)
  - Client CRM
  - Projects & tasks
  - Invoices
  - Goals & dashboard overview

  The original design reference is available on Figma: https://www.figma.com/design/D6i7PSBgqJsxSD523fFOdL/Design-Solopreneur-OS-App.

  ## Running the app

  From the project root:

  1. Install dependencies (using pnpm):

     ```bash
     pnpm install
     ```

  2. Start the development server:

     ```bash
     pnpm dev
     ```

  The app will be available at the URL printed in the terminal (typically `http://localhost:5173` or similar).
